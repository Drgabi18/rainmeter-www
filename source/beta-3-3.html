Rainmeter 3.3 Beta

The folowing are the changes for the 3.3 beta version of Rainmeter. You can safely install the new 3.3 beta version right over top of your existing Rainmeter 3.2 or earlier installation, and nothing you have done will be lost or changed. There is never any need to uninstall any earlier version of Rainmeter to use the newest version.
