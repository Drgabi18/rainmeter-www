Rainmeter 3.2 Beta

There are a lot of new features and capabilities in the latest Rainmeter 3.2. You can safely install the new 3.2 beta version right over your existing Rainmeter 3.1 installation, and nothing you have done will be lost or changed. There is never any need to uninstall any earlier version of Rainmeter to use the newest version.
