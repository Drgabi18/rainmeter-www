Rainmeter 3.0.1 Release

Time for the next version of Rainmeter!

The latest version of Rainmeter will see many improvements, both for users of the application and for those creating and editing skins. Stay tuned every Sunday for the latest beta version. As always, Rainmeter 3.1 fully supports skins written for earlier versions, and users can simply upgrade their existing installations. There is no need to uninstall or change any existing setup to use the new beta version.

November 10, 2013 - Revision 2187

  • Added Measures: New IfConditions functionality added. This is a powerful extension to the concepts provided in IfActions, allowing multiple tests and actions, both "true" and "false" testing, and the ability to use logical AND and OR operators in the conditions. See Change Announcements for details and example skins.
